### Sobre o Projeto
Este projeto foi desenvolvido durante a formação gratuita da Xperiun, com o objetivo de analisar o desempenho comercial de uma empresa fictícia. O foco foi transformar dados brutos em insights estratégicos para tomada de decisão gerencial.

### Tecnologias e Ferramentas
* Power BI: Construção do dashboard e visualização de dados.
* DAX (Data Analysis Expressions): Utilizado para o tratamento de dados e criação de medidas calculadas.
* Modelagem de Dados: Estruturação de tabelas e dimensão.
* Power Point: Design de visualização do fundo do dashboard.

### Inteligência do Negócio (DAX)
Neste projeto, optei por realizar o tratamento e a criação de métricas utilizando a Linguagem DAX, focando em performance e flexibilidade. Algumas das análises implementadas foram:
* Cálculos de Acumulados (YTD) para acompanhamento de metas.
* Medidas de Performance: Faturamento total, quantidade vendida e ticket médio.
* Análises Comparativas: Comparação de vendas entre diferentes períodos e supervisores.

### Insights
O dashboard permite identificar:
* Performance por Supervisor/Vendedor: Visualização clara de quem está liderando os resultados.
* Top Produtos: Identificação dos itens com maior volume de saída e custo unitário.
* Sazonalidade: Análise mensal das vendas para identificar picos e vales de demanda.

### Conclusão
O projeto conclui o ciclo completo de Business Intelligence: extração, tratamento de dados fragmentados e visualização. O uso avançado de linguagem DAX permitiu criar métricas dinâmicas que garantem uma análise precisa de sazonalidade e performance de vendas. A estrutura de dados montada suporta a escalabilidade para novos meses, servindo como uma ferramenta robusta de monitoramento de KPIs em tempo real.
